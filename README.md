# Quant UX Research Academy

An interactive learning platform to help you become an expert quantitative UX researcher and data scientist.

## Features

- **4 Learning Paths**: Survey Research, Statistical Analysis, Programming & Tools, Research Design
- **20+ Interactive Lessons**: Comprehensive content with examples and resources
- **Progress Tracking**: Your progress is saved locally in your browser
- **Mobile-Responsive**: Optimized for learning on any device
- **Practice Exercises**: Test your knowledge with interactive quizzes
- **External Resources**: Links to deeper learning materials

## Getting Started Locally

1. Open `index.html` in your web browser
2. Start learning! Your progress is automatically saved

That's it! No build process or dependencies needed.

## Deploying Online

You have several free hosting options:

### Option 1: GitHub Pages (Recommended)

1. Create a new GitHub repository
2. Upload all files to the repository
3. Go to Settings > Pages
4. Select "main" branch as source
5. Your site will be live at `https://yourusername.github.io/repository-name`

### Option 2: Netlify

1. Go to [netlify.com](https://netlify.com)
2. Sign up for free account
3. Drag and drop your project folder
4. Your site is instantly live!
5. Get a custom domain or use the free `.netlify.app` subdomain

### Option 3: Vercel

1. Go to [vercel.com](https://vercel.com)
2. Sign up for free account
3. Click "New Project" and import your files
4. Deploy in seconds

### Option 4: Cloudflare Pages

1. Go to [pages.cloudflare.com](https://pages.cloudflare.com)
2. Connect your GitHub repository
3. Deploy automatically

## How to Use

2. **Learn**: Access 20+ lessons organized by topic
3. **Practice**: Test your knowledge with exercises
4. **Challenges**: Future games and advanced challenges

## Progress Tracking

Your progress is stored in your browser's localStorage. This means:
- ✅ No account required
- ✅ Works offline after first visit
- ✅ Fast and private
- ⚠️ Clearing browser data will reset progress
- ⚠️ Progress doesn't sync across devices
